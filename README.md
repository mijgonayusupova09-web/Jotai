## Jotai State Management Examples

This repository contains examples of atomic state management using Jotai. It covers primitive atoms, derived atoms, and asynchronous atoms for data fetching and side effects.

### Features

*   Atom-based state modeling
*   Derived atoms for computed state
*   Async atoms with async getters/setters
*   Component-level granularity
*   Clean and minimal React integration

### Tech Stack

*   Vite
*   Jotai
*   React-Router
